\begin{abstract}
哈希表是计算机领域最基础的核心数据结构，近六十年来，如何在操作时间与存储空间之间取得最优权衡，一直是数据结构方向的核心研究难题。本文以《On the Optimal Time/Space Tradeoff for Hash Tables》为研究基础，对论文中的核心算法原理、工程实现细节进行梳理，并尝试简化实现论文中描述的哈希系统。

论文中包含三部分核心工作，第一是复现 K - kick Tree 球槽映射及分层驱逐这种核心算法；第二是设计并实现 MiniArray 与本地查询路由两类紧凑元数据结构，完成变长路由信息的低开销存储与常数时间访问；第三则是在此基础上搭建增强开放寻址哈希表整体架构，实现插入、删除、查找、动态扩容及商压缩优化机制，完整复现论文设计逻辑。

最终实现常数时间查询，插入与删除时间高概率可控制，而且具备动态扩容与缩容的精简哈希表系统。经实验数据的结果来看，该系统再现了原论文当中有关时间和空间兼顾的特点，证实了这种较为理想的哈希结构在实际工程中有得以落实的可能性，进而给高性能精简哈希表的设计及其应用赋予一定的参照。
\end{abstract}

\begin{abstract*}
Hash tables are the most fundamental core data structures in the field of computer science. For nearly sixty years, how to achieve the optimal balance between operation time and storage space has been a central research problem in the study of data structures. This paper is based on "On the Optimal Time/Space Tradeoff for Hash Tables," and it organizes the core algorithm principles and engineering implementation details presented in the paper, while attempting to simplify the implementation of the hash system described.

The paper contains three core contributions. The first is reproducing the key algorithm of K-kick Tree slot mapping and hierarchical eviction; the second is the design and implementation of two types of compact metadata structures, MiniArray and local query routing, which enable low-overhead storage of variable-length routing information and constant-time access; the third is building an enhanced open-addressing hash table architecture based on this, implementing insertion, deletion, lookup, dynamic resizing, and quotient compression optimization mechanisms, fully reproducing the design logic of the paper.

The final implementation achieves constant-time queries, and the insertion and deletion times are highly probable to be controlled, while providing a streamlined hash table system with dynamic expansion and contraction. According to experimental results, this system replicates the time and space tradeoff characteristics described in the original paper, demonstrating that this relatively ideal hash structure can be realized in practical engineering, thus providing a reference for the design and application of high-performance streamlined hash tables.
 \end{abstract*}
